2. Non-Primitive (Reference) Data Types
->Non-primitive types are created by the programmer and include classes, interfaces, arrays, and enums. 
->These types reference objects or instances, and they don't have fixed sizes.

String: A sequence of characters. Example: String str = "Hello";
Arrays: Used to store multiple values of the same type. Example: int[] arr = {1, 2, 3};
Classes and Objects: Classes define the structure and behavior of objects. Example: Person person = new Person();
Interfaces: Abstract types used to specify a behavior that classes must implement.
